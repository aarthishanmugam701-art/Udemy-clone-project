# Udemy Clone – HTML & CSS Project

This project is a Udemy website clone created using HTML5 and CSS3 as part of my frontend development practice. The website replicates the basic layout and structure of the Udemy platform.

## Features

- Navigation bar with logo and search bar  
- Course categories section  
- Recommended courses section  
- Course cards with image, title, rating, and price  
- Responsive layout using CSS Flexbox  

##  Technologies Used

- HTML5  
- CSS3  

##  Project Structure

- index.html – Main homepage  
- style.css – Styling file  
- images/ – Contains all images used in the project  

##  Objective

The objective of this project is to strengthen my understanding of HTML structure, CSS styling, layout design, and responsive web development.
